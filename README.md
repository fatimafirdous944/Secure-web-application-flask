# Building a Secure Web Application using Python Flask

This project demonstrates the development of a secure web application using Python Flask with user authentication and security best practices.

## Features
- Secure User Registration and Login
- Password Hashing
- Session Management
- Protected Routes
- Secure Backend Logic

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Tailwind CSS

## Cybersecurity Concepts Implemented
- Authentication and Authorization
- Secure Password Storage
- Session Security
- Input Validation

## Purpose
This project was created for learning secure web application development and for internship demonstration purposes.
